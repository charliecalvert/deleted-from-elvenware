Talk 3312

Leveraging the Power and Simplicity of SQL Server with JBuilder

Alain Tadros and Charlie Calvert — Falafel Software, Inc.

Description

This session demonstrates how to use SQL Server as a database platform from JBuilder 9. The session primarily uses SQL Server 2000 and the Microsoft JDBC Driver for building datacentric applications. However, the session also gives some information on how the next version of SQL Server, code named "Yukon", provides more interoperability options by supporting connectivity through Web Services.

Obtaining the driver.

  • If you register for a free .NET Passport then the driver is free and you can download it from the Microsoft site.
  • All things MS SQL are found at http://www.microsoft.com/sql/.
  • The dirver, at the time this was written, is here: http://www.microsoft.com/sql/downloads/jdbcregister.asp
