Description: This program is discussed in Chapter 23, Simple Applications. The program outputs a string to the screen at the command line. It is meant to be the simplest possible program you can write in Java. I show this program not as a lesson in how Java works, but to show you how to get simple programs of this type running in the IDE. Read Chapter 23 for a discussion of that topic.

Here is the source:

public class Java101
{
  public static void main(String[] args)
  {
    System.out.println("Turn, turn, to the wind and the rain.");
  }
}

You will be able to run this from the command line with Ant.

You could also first compile and then run it by issuing the following commands:

javac Java101.java
java -cp . Java101

The scripts run.bat and run.sh should allow you to run the program from the command line.
