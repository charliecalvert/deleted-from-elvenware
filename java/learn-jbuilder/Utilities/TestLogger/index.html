Description: This program only compiles with JDK 1.4 or greater.

TestLogger demonstrates how to use the Java logging facilities that first appeared in JDK 1.4. It is discussed in Chapter 29, on debugging.

Because it doesn't compile with the JDK that comes with JBuilder 7, I have not put it in the ant build process that is found on the CD.
