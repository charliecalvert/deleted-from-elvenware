You can turn on assertions for JDK 1.3.1 in JBuilder by turning to the General Page

Command line and Ant issues

Pass source="1.4" to javac when compiling from the command line ot enable assertions.

Pass -ea to java to turn the assertions on.

Here is how to pass handle assertions this in Ant:

		<javac srcdir="${src}" destdir="${build}" source="1.4"/>

		<java classname="${small-name}.${caps-name}" fork="yes">
			<jvmarg value="-ea"/>
			<classpath>
				<pathelement location="${project-jar}"/>
				<pathelement path="${java.class.path}"/>
			</classpath>
		</java>
