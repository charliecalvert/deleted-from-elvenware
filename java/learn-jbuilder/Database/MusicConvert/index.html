Project: MusicConvert
Author: Charlie Calvert
Company: Elvenware
Help: index.html

Description:

This program is discussed in Chapter 39, on database setup, and in chapter 41, on database basics. It's purpose is to convert data from InterBase or MySQL format to simple text based files. The files can be either filled with SQL statements, or they can be filled with commad delimited data. The data is written automatically to a subdirectory of your home directory called elvenware:

  • c:\documents and settings\user name\elvenware
  • \home\user name\elvenware

By pressing the Text Convert button you ask the program to convert data to comma delimited format using the Borland database tools. There is a schema file associated with the text file that describes the fields and their names. These files can be read back in and converted into normal database tables. See the ConvertToSQL.java file in the com.elvenware.database package. The program is interesting because it reads the metadata from a table.

If you have trouble running this program, index.html contains useful information about setting up the code that comes with this book. Setting up databases is described at lengh in the first few chapters of the database section of the book.

This program relies on an InterBase database called music2.gdb. It ships on the CD that comes with this book in the /Database/Data directory. You should place this file in a directory called data which is beneath the home directory on your system. For windows, the home directory is usually in c:\Documents and Directories\UserName. On Linux, it is in the /home/UserName. For instance, the file should be placed in:

/home/ccalvert/data/music2.gdb
