Author: Charlie Calvert
Company: Elvenware
Help: index.html

Description: This is the example program for the Database Techniques chapter. It demonstrates two different JBuilder technologies for creating a one to many relationship between two tables. The chapter ends by covering a simple example of a calculated field. Code showing how to edit the calculated field is also included.

The chapter begins by showing how to create a one to many with the data modeler. It then shows how to build a one to many from scratch inside a data module.

This program includes an examples of an "enumerated class" type in the FirstLastInput dialog.

This program is discussed in Chapter 42, on database techniques.
