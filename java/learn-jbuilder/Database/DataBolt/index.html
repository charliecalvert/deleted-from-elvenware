Project: DataBolt
Author: Charlie Calvert
Company: Elvenware
Help: index.html

Description: The DataBolt project is a simple data browser for MySQL Tables. Use it to view all the databases on your system, and each of the tables inside those databases.

By default, the database is "jdbc:mysql://localhost/," but there is an option in the file menu that lets you edit this at run time. At design time, you can change this value in the Globals.java file.

If you are working against a Linux server, then you should use case sensitive queries.
