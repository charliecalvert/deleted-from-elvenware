Description:

You can recreate all the data for the music portion of the jbbook database.

The script create-music.sh will create the tables. The script delete-music.sh will delete the tables. Go to the data directory and run commandset.sh to to put the data into the tables.

I somehow got my socket privileges messed and up could not connect to mysql from a local machine as anyone but root because of an error that looked like this: "Can't connect to local MySQL server through socket '/var/lib/mysql/mysql.sock'." I fixed this problem by using the host option, which connects through tcpip rather than through a server: mysql --host=bree --user=charlie --password=foo
