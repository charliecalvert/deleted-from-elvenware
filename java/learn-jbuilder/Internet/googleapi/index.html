Project: Google Api
Author: Charlie Calvert
Company: Elvenware
Help: index.html

Description: An example of how to use the Google APIs to run queries again Google from your java programs. Actually, its a web service, so you could do it from anywhere.

All the code you need is present and ready to use. At run time, however, you will be prompted for your google license key. You need to sign on to google's site to get a key./ At the time of this writing the place to go was:

http://www.google.com/apis/

The sign in process is free, and they will immediately mail you your key.

If you have forgotten your key, sign in again and they will resend your key.

This progarm is not mentioned in the book. The program works fine, and the topic is interesting, but the book was too long already, and the Google API is not really closely related to JBuilder itself.
