Project: Open Tools Basic GUI App
Author: Charlie Calvert
Company: Elvenware
Help: index.html

Description: This program pops up a dialog. Push the Ok button in the dialog, and the dialog disappears and a message is placed in the Message pane.

This program is not discussed in the Chapter on OpenTools. You do not run this program, instead, it creates a jar file which becomes part of the JBuilder IDE. Read the chapter for further information on open tools.
