Project: OpenToolsBasic
Author: Charlie Calvert
Company: Elvenware
Help: index.html

Description: Create a simple open tool for inclusion in JBuilder itself

This program is discussed in Chapter 32 Open Tools

If you have trouble running this program, index.html contains useful information about setting up the code that comes with this book.

You might be interested in the ant file for compiling this project, since it shows how to designate a new classpath when compiling:

		<javac srcdir="${src}" destdir="${build}">
			<classpath>
				<pathelement path="${java.class.path}/"/>
				<pathelement path="${jbuilder-lib}/jbuilder.jar"/>
			</classpath>
		</javac>

 

Here is the classes.opentools file:

OpenTools-Core: opentoolsbasic.SimpleOpenTools

Don't forget the final carriage return!
