This program is discussed in Chapter 25, the Complex Projects chapter.

This is a particular interesting example because of what is happening with the classpath. As explained at length in the book, the ElfTimer.java file which is key to this project is stored in the src directory beneath this directory. In particular, it is outside the directory that hosts this project. One of the main points of the Complex Projects chapter is to explain how to handle this kind of situation inside of JBuilder. Here is code for handling this problem in an Ant script:

		<javac destdir="${build}">
			<src path="${src}"/>
			<src path="$../src"/>
		</javac>

Notice that the second src element points at the directory that holds the timeit.ElfTimer.java component.
